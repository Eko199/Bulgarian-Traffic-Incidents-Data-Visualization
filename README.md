# Bulgarian Traffic Incidents Open Data Visualization
A lightweight Streamlit app for exploring Bulgaria’s public traffic-incident data. It loads open datasets, visualizes trends and hotspots, and offers quick, interactive tools for filtering and analyzing road-safety patterns.

### How to start:
1. `pip install -r requirements.txt`
2. `streamlit run app.py`

### Sources:
- Traffic incidents data: https://data.egov.bg
- Analog clocks with matplotlib: https://github.com/ibskhan4/clocks
- Bulgarian provinces GeoJSON: https://github.com/yurukov/Bulgaria-geocoding

# Визуализация на отворени данни за брой ПТП в България
Web приложение на Streamlit за проучване на данни за пътни инциденти в България. То зарежда отворени набори от данни, визуализира тенденции и предлага бързи, интерактивни инструменти за филтриране и анализ на модели на пътна безопасност.

### Стартиране:
1. `pip install -r requirements.txt`
2. `streamlit run app.py`

### Източници:
- Данни за ПТП: https://data.egov.bg
- Аналогови часовници с matplotlib: https://github.com/ibskhan4/clocks
- GeoJSON на българските области: https://github.com/yurukov/Bulgaria-geocoding
