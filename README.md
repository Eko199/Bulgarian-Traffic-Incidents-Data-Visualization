# Bulgarian Traffic Incidents Data Visualization
A lightweight Streamlit app for exploring Bulgaria’s public traffic-incident data. It loads open datasets, visualizes trends and hotspots, and offers quick, interactive tools for filtering and analyzing road-safety patterns.

**Стартиране**:
`streamlit run app.py`

**Източници**:
- Аналогови часовници с matplotlib: https://github.com/ibskhan4/clocks
- Bulgarian provinces GeoJSON: https://github.com/yurukov/Bulgaria-geocoding
